# Duckov-Mod---useQchangeweapon
《逃离鸭科夫》模组 按Q切换至最近武器
https://steamcommunity.com/sharedfiles/filedetails/?id=3597205849

模拟 CS 和 CF 的切枪习惯，按“Q键”切换至最近使用武器，设想是 "切狙"、“切刀” 或 “副手补枪”

目前支持 “主武器、副武器、近战武器” 之间相互切换

一定程度上令大狙或重武器跑路手感不再笨重

频繁用于取消武器后摇将影响游戏平衡 awa

提示：配合“击杀特效”等模组体验更佳

目前已知问题：从道具栏（比如手雷等）按Q会切错武器

2025.11.2：模组重写，直接使用游戏内键位，仅Q键无法更改

2025.11.3：手感优化，解决“当玩家处于无法切换武器时按下Q键，让后续Q键卡手”的问题。
移除本mod中 大量无关紧要的Debug日志生成
