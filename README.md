# Duckov-Mod---useQchangeweapon
《逃离鸭科夫》模组 按Q切换至最近武器
https://steamcommunity.com/sharedfiles/filedetails/?id=3597205849

## 模拟 CS 和 CF 的切枪习惯，按“Q键”切换至最近使用武器，如“ 大狙 →Q→ 刀 →Q→ 大狙 ”
目前支持 “主武器、副武器、近战武器、道具（如手雷、卡祖笛等）” 的相互切换

通过玩家自己的操作，强化手持笨重武器时的灵活性

频繁用于取消武器后摇将影响游戏平衡 awa

## 本人代码基础约等于0，命名不规范等坏习惯请勿学习

思路：订阅 CharacterInputControl 的 inputaction 回调判断玩家行为，记录数值进行对调后使用 CharacterMainControl.Main 控制玩家角色。

检测输入用的方法不算好，还将初始化扔进了update里。操作很粗暴，但颇有乱拳打死老师傅的快感。
